## Descripción

Este laboratorio tiene como objetivo aplicar conceptos de **programación funcional y orientada a objetos en Python**.  
Se implementan ejercicios prácticos para consolidar el uso de:

- Generadores con `yield`
- Iteradores con `iter()`
- Listas por comprensión
- Funciones anónimas con `lambda`
- Transformaciones con `map()`
- Filtrado de datos con `filter()`
- Reducción de colecciones con `reduce()`

Cada grupo de ejercicios se encuentra organizado en carpetas independientes, según el tipo de función utilizada.  
Cada archivo `.py` contiene código limpio, comentado y funcional.
